### Usage

* import ``enum.enum`` first
* Use ``Enum.new(X)`` to create a Enum Object (with initialize-value ``X``)
* Use ``<EnumObject>.enum()`` to get enum-value
* Use ``<EnumObject>.reset()`` to reset this object
